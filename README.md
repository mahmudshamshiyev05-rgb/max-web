# MAX Web

MAX is a voice-first AI assistant focused on natural communication in Uzbek, Russian, and English.

## Vision

MAX aims to become a personal AI assistant that communicates naturally, remembers important information, and helps users through voice interaction.

## Status

Project is currently in active development.

## Features (MVP)

- Voice Input
- Voice Output
- Multi-language Support
- Conversation Context
- AI-powered Responses
